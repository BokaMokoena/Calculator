# Calculator
# Project Description
This is a simple console based Java program that allows users to insert numbers then it performs basic arithmetic operations (add, subract, divide and mutliple). This is developed using OOP in Java where Calculator class handles the calculations and Main class runs the program and manages the user interaction.

# Features
- Basic Arithemetic calculations
- Error handling if invalid input is inserted
- Menu-driven console interface
- Runs until user chooses to exit

# How to run the project
1. Clone the Repository
git clone (https://github.com/BokaMokoena/Calculator)

2. Navigate to the Source Folder

cd Calculator/src

3. Compile the Program

javac Main.java Calculator.java

4. Run the Program
